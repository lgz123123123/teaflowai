TeaFlow is a Zen-inspired, minimalist tea brewing companion web application designed specifically for **Sencha** and **Gyokuro**. It guides users through the intricate process of multi-steep brewing, adhering to Japanese tea ceremony principles.

## Features

- **Aesthetic Design**: Neumorphism meets Wabi-sabi with a calming Sage/Moss color palette.
- **Master-Level Brewing**: Variable temperature curves for each steep (e.g., lower temp for Umami extraction, higher for aroma).
- **Interactive Timer**: A fluid, ripple-effect timer that persists even when the tab is in the background.

## Getting Started

1.  **Install dependencies**
    ```bash
    npm install
    ```

2.  **Run development server**
    ```bash
    npm run dev
    ```

3.  **Build for production**
    ```bash
    npm run build
    ```
